$ composer require <library>

NPM

npmjs.org

Need a JavaScript module?

$ npm install <module>

Don’t repeat yourself.

Don’t manage 3rd party code.

Just declare required components and versions.

Manages dependencies and conflicts.

Browserify

browserify.org

Browserify lets you use NPM modules in the browser.

Step 1

$ npm install uniq

Step 2

var unique = require('uniq'),
    data   = [1, 2, 2, 3, 3, 3, 4];
    
unique(data); // => [1, 2, 3, 4]

Asset Building

Sass

sass-lang.com

Sass is a better way to write CSS.

CSS

.content h1,
.content h2,
.content h3,
.content p {
  color: black;
}

Sass/SCSS

$text-color: black;

.content {
  h1, h2, h3, p {
    color: $text-color;
  }
}
